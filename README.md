# dotNetCore-RestAPI
A basic .Net Core API code


This is the basic REST API calls in .NetCore with SSMS (Sql Server Management Studio) with Migration files.

Get, Post, Put, Delete

Note: Run > dotnet ef database Update


# API Commands

Get: api/comands

Get: api/comands/n

Post: api/comands

Put: api/comands/n

Delete: api/comands/n
